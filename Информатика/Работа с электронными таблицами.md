Электронная таблица---это компьютерная программа, позволяющая хранить, организовывать, анализировать и представлять данные в виде таблиц, состоящих из строк и столбцов. 

## Google Sheets
Сегодня мы поговорим о Google Sheets. Он поддерживает сотню функций для расчётов, анализа данных и автоматизации. Формула начинается со знака =. А функции - это встроенные команды, такие как СУММ (SUM), ЕСЛИ (IF). Полный список функций доступен через справку.

## Формулы
| Английское название функции | Русское название функции (Excel) | Назначение функции |
| :--- | :--- | :--- |
| SUM() | СУММ() | Вычисляет сумму числовых значений в диапазоне ячеек. |
| AVERAGE() | СРЗНАЧ() | Вычисляет среднее арифметическое числовых значений в диапазоне ячеек. |
| MIN() | МИН() | Находит минимальное (наименьшее) числовое значение в диапазоне ячеек. |
| MAX() | МАКС() | Находит максимальное (наибольшее) числовое значение в диапазоне ячеек. |
| COUNT() | СЧЁТ() | Подсчитывает количество ячеек, содержащих числа, в диапазоне. |
| IF() | ЕСЛИ() | Проверяет выполнение условия и возвращает одно значение, если условие истинно, и другое, если ложно. |
| AND() | И() | Возвращает ИСТИНА, если все логические аргументы имеют значение ИСТИНА; в противном случае возвращает ЛОЖЬ. |
| OR() | ИЛИ() | Возвращает ИСТИНА, если хотя бы один из логических аргументов имеет значение ИСТИНА; в противном случае возвращает ЛОЖЬ. |
| COUNTIF() | СЧЁТЕСЛИ() | Подсчитывает количество ячеек в диапазоне, удовлетворяющих заданному критерию. |
| VLOOKUP() | ВПР() | Ищет значение в первом столбце диапазона и возвращает значение из ячейки в той же строке из указанного столбца. (Вертикальный Просмотр) |
| HLOOKUP() | ГПР() | Ищет значение в первой строке диапазона и возвращает значение из ячейки в том же столбце из указанной строки. (Горизонтальный Просмотр) |
| INDEX() | ИНДЕКС() | Возвращает значение ячейки, находящейся на пересечении указанной строки и указанного столбца в пределах заданного диапазона. |
| MATCH() | ПОИСКПОЗ() | Ищет указанное значение в одномерном диапазоне (строке или столбце) и возвращает относительную позицию этого значения в диапазоне. |
| CONCATENATE() | СЦЕПИТЬ() | Объединяет несколько текстовых строк в одну. |
| LEN() | ДЛСТР() | Возвращает количество символов в текстовой строке. |
| DATE() | ДАТА() | Возвращает порядковое число, соответствующее указанной дате, состоящей из года, месяца и дня. |
| TODAY() | СЕГОДНЯ() | Возвращает текущую дату в формате даты. |
| SORT() | СОРТ() | Сортирует диапазон или массив по значениям одного или нескольких столбцов. *(Доступна в современных версиях Excel и Google Sheets)* |
| FILTER() | ФИЛЬТР() | Фильтрует диапазон или массив данных на основе заданных условий. *(Доступна в современных версиях Excel и Google Sheets)* |
| IMPORTRANGE() | IMPORTRANGE() | *Функция Google Sheets.* Импортирует данные из указанного диапазона ячеек из другой таблицы Google. (В Excel нет прямого аналога с тем же названием). |
